# VTEX Service Tax Middleware - Exención IVA San Andrés

Middleware desarrollado para la integración con VTEX Service Tax, encargado de calcular y aplicar la exención de IVA para órdenes con destino a San Andrés.

## Tecnologías
- VTEX IO
- Node.js
- TypeScript
- Service Tax API

## Objetivo
Calcular el beneficio tributario y reflejarlo en el checkout de VTEX antes de finalizar la compra.
